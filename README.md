### 1.项目演示地址

#### 1.1 PC端：[iLoveHotel](http://lbbjzz.com)

#### 1.2 移动端: [iLoveHotel_H5](http://h5.lbbjzz.com)

### 2.Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:5200
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
